# Friends Chat Telegram Assistant

Телеграм-бот для групповых чатов, который отвечает через OpenAI Assistants API и помнит контекст диалога внутри каждой группы.

## Что умеет бот
- Отвечает только в группе/супергруппе.
- Реагирует только когда:
  - его упомянули через `@username`;
  - вы ответили реплаем на сообщение бота.
- Ведет отдельный постоянный контекст по каждой группе:
  - для каждого `chat_id` хранится свой `thread_id` в SQLite;
  - после перезапуска история для группы не теряется.
- Если ответ от OpenAI идет дольше 20 секунд, бот показывает статус "печатает...".
- Ошибки и таймауты обрабатываются, логи пишутся в `bot.log`.

## Как пользоваться в обычном чате
1. Добавьте бота в группу.
2. Убедитесь, что у бота есть право читать сообщения.
3. Обращайтесь к нему одним из способов:
   - `@ваш_бот объясни простыми словами что такое RAG`
   - реплай на ответ бота: `а теперь короче`

Бот игнорирует все сообщения без упоминания и без реплая на его сообщение.

## Примеры запросов
- `@ваш_бот составь план на неделю тренировок`
- `@ваш_бот проверь этот текст на ошибки: ...`
- реплай на бота: `сделай ответ в 3 пунктах`
 
## Утренний обзор
- Добавьте `NEWS_CHAT_IDS`, чтобы перечислить ID чатов, куда нужно отправлять обзор каждое утро.
- Это работает на основе OpenAI Assistant: каждый чат сохраняет свой `thread_id`, и в 09:00 бот запускает `NEWS_PROMPT`.
- `NEWS_PROMPT` по умолчанию — `Составь короткий утренний обзор новостей (политика, экономика, технологии) в 3–4 пунктах.` Можно адаптировать под стиль группы.
- Время отправки можно скорректировать через `NEWS_TIMEZONE` (по умолчанию `Europe/Moscow`). Если нужно другое время, установите нужный часовой пояс и измените текст в `NEWS_PROMPT`.

## Переменные окружения
Скопируйте `.env.example` в `.env` и заполните:
- `TELEGRAM_BOT_TOKEN`
- `OPENAI_API_KEY`
- `OPENAI_ASSISTANT_ID`
- `OPENAI_VECTOR_STORE_ID` (опционально, если vector store уже привязан в Assistant)

## Быстрый старт
1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите бота из папки `bot/`:
   ```bash
   python main.py
   ```

## Структура проекта
- `main.py` - запуск бота, Telegram handlers, фильтрация сообщений
- `assistant.py` - обертка над OpenAI Assistants API
- `database.py` - SQLite-хранилище `chat_id -> thread_id`
- `.env.example` - пример переменных окружения
- `requirements.txt` - зависимости

## Тех. детали
- Стек: `python-telegram-bot` (v20+), `openai` SDK (v1+), `sqlite3`.
- Используется Assistants API (`beta.threads` / `beta.runs`), не Chat Completions.
- При старте бот синхронизирует настройки ассистента (`temperature`, `file_search`).
